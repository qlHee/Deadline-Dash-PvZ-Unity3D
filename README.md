# Deadline-Dash-PvZ-Unity3D

This game creatively combines the classic IP Plants vs. Zombies with a parkour game, bringing various plants onto the parkour path to make the game tense and exciting, allowing players to easily get started. By integrating various parkour characters and game modes, it elevates the game's playability to a peak.

## Controls

- **Accelerate / Decelerate**: W / S or Arrow Keys
- **Shift Left / Right**: A / D or Arrow Keys
- **Jump**: Spacebar  
- **Attack**: Shift or X
- **Pause**: ESC

## Game Versions

**Game source code:**
- **PureRUN-1** to **PureRUN-10**: Mainly developing iterative version, no puzzle part
- **ZomRun-1**: Complete version including the puzzle

**Game executable program:**
- **macOS-ZomRun-pure**: A pure parkour game developed for Mac (no puzzles)
- **MacOS-ZomRun**: Complete game developed for Mac
- **Windows-ZomRun-pure**: A pure parkour game developed for Windows (no puzzles)

## Setup

**Run the source code:**
1. Clone this repository
2. Open project in Unity 2022.3 LTS
3. Open `Assets/Scenes/Main Scene.unity` and press Play

**Run the executable program:**
- **MacOS-ZomRun**: Open Run.app
- **macOS-ZomRun-pure**: Open ZomRun-pure.app
- **Windows-ZomRun**: open RUN-Main/RUN-1.exe
- **Windows-ZomRun-pure**: open ZomRun.exe
